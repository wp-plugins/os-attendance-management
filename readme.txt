== Description ==

出勤・退勤などの勤怠が管理できるプラグインです。
出退勤を一覧でき、データをCSVとしてダウンロードできます。


== Installation ==

1. `/wp-content/plugins/`ディレクトリにアップロードします。
   Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. メニューの'プラグイン'で一覧を表示し、出勤・勤怠管理プラグインを有効化します。
   Activate the plugin through the ‘Plugins’ menu in WordPress
